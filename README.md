# Python-selenium

파이썬 셀레니움 연습하기

### 가상환경 진입

python -m venv selenium

### 가상환경 실행

cd selenium\Scripts 진입후
activate

### 셀레니움 다운로드 및 설치

pip install selenium

### 크롬 드라이버 다운로드

https://chromedriver.chromium.org/downloads

버전에 맞는걸로 다운로드 후 chromedriver.exe 파일을 폴더에 붙이기
